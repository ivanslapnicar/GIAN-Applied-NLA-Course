Norms

In order to measure changes, we need to define norms. For more details and the proofs of the Facts below, see R. Byers and B. N. Datta, Vector and Matrix Norms, Error Analysis, Efficiency, and Stability, pp. 50.1-50.24 and the references therein.

Definitions

Norm on a vector space X is a real-valued function ∥x∥:X→R with the following properties:

  1. Positive definiteness. ∥x∥≥0 and ∥x∥=0 if and only if x is the zero vector.

  2. Homogeneity. ∥λx∥=|λ|∥x∥

  3. Triangle inequality. ∥x+y∥≤∥x∥+∥y∥

Commonly encountered vector norms for x∈Cn are:

  • Hölder norm or p-norm: for p≥1, ∥x∥p=(|x1|p+|x2|p+⋯|xn|p)1/p,

  • Sum norm or 1-norm: ∥x∥1=|x1|+|x2|+⋯|xn|,

  • Euclidean norm or 2-norm: ∥x∥2=|x1|2+|x2|2+⋯|xn|2,

  • Sup-norm or ∞-norm: ∥x∥∞=maxi=1,…,n|xi|.

Vector norm is absolute if ∥|x|∥=∥x∥.

Vector norm is monotone if |x|≤|y| implies ∥x∥≤∥y∥.

From every vector norm we can derive a corresponding induced matrix norm (also, operator norm or natural norm):

∥A∥=maxx≠0∥Ax∥∥x∥=max∥x∥=1∥Ax∥.

For matrix A∈Cm×n we define:

  • Maximum absolute column sum norm: ∥A∥1=max1≤j≤n∑i=1m|aij|,

  • Spectral norm: ∥A∥2=ρ(A∗A)=σmax(A) (largest singular value of A),

  • Maximum absolute row sum norm: ∥A∥∞=max1≤i≤m∑j=1n|aij|,

  • Euclidean norm or Frobenius norm:

∥A∥F=∑i,j|aij|2=tr⁡(A∗A)

.

Matrix norm is consistent if ∥A⋅B∥≤∥A∥⋅∥B∥, where A and B are compatible for matrix multiplication.

Matrix norm is absolute if ∥|A|∥=∥A∥.

Facts

  1. ∥x∥1, ∥x∥2, ∥x∥∞ and ∥x∥p are absolute and monotone vector norms.

  2. A vector norm is absolute iff it is monotone.

  3. Convergence. xk→x∗ iff for any vector norm ∥xk−x∗∥→0.

  4. Any two vector norms are equivalent in the sense that, for all x and some α,β>0

α∥x∥μ≤∥x∥ν≤β∥x∥μ.

In particular:

  • ∥x∥2≤∥x∥1≤n∥x∥2,

  • ∥x∥∞≤∥x∥2≤n∥x∥∞,

  • ∥x∥∞≤∥x∥1≤n∥x∥∞.

  1. Cauchy-Schwartz inequality. |x∗y|≤∥x∥2∥y∥2.

  2. Hölder inequality. If p,q≥1 and 1p+1q=1, then

|x∗y|≤∥x∥p∥y∥q.

  1. ∥A∥1, ∥A∥2 and ∥A∥∞ are induced by the corresponding vector norms.

  2. ∥A∥F is not an induced norm.

  3. ∥A∥1, ∥A∥2, ∥A∥∞ and ∥A∥F are consistent.

  4. ∥A∥1, ∥A∥∞ and ∥A∥F are absolute. However, ∥|A|∥2≠∥A∥2.

  5. Any two matrix norms are equivalent in the sense that, for all A and some α,β>0

α∥A∥μ≤∥A∥ν≤β∥A∥μ.

In particular:

  • 1n∥A∥∞≤∥A∥2≤m∥A∥∞,

  • ∥A∥2≤∥A∥F≤n∥A∥2,

  • 1m∥A∥1≤∥A∥2≤n∥A∥1.

  1. ∥A∥2≤∥A∥1∥A∥∞.

  2. ∥AB∥F≤∥A∥F∥B∥2 and ∥AB∥F≤∥A∥2∥B∥F.

  3. If A=xy∗, then ∥A∥2=∥A∥F=∥x∥2∥y∥2.

  4. ∥A∗∥2=∥A∥2 and ∥A∗∥F=∥A∥F.

  5. For a unitary matrix U of compatible dimension,

∥AU∥2=∥A∥2,∥AU∥F=∥A∥F,∥UA∥2=∥A∥2,∥UA∥F=∥A∥F.

  1. For A square, ρ(A)≤∥A∥.

  2. For A square, Ak→0 iff >ρ(A)<1.

Errors and condition numbers

We want to answer the question:

How much the value of a function changes with respect to the change of its argument?

Definitions

For function f(x) and argument x, the absolute error with respect to the perturbation of the argument δx is

∥f(x+δx)−f(x)∥=∥f(x+δx)−f(x)∥∥δx∥∥δx∥≡κ∥δx∥.

The condition or condition number κ tells how much does the perturbation of the argument increase. (Its form resembles derivative.)

Similarly, the relative error with respect to the relative perturbation of the argument is

∥f(x+δx)−f(x)∥∥f(x)∥=∥f(x+δx)−f(x)∥⋅∥x∥∥δx∥⋅∥f(x)∥⋅∥δx∥∥x∥≡κrel∥δx∥∥x∥.

Peturbation bounds

Definitions

Let A∈Cn×n.

Pair (λ,x)∈C×Cn×n is an eigenpair of A if x≠0 and Ax=λx.

Triplet (y,λ,x)∈×Cn×C×Cn is an eigentriplet of A if x,y≠0 and Ax=λx and y∗A=λy∗.

Eigenvalue matrix is a diagonal matrix Λ=diag⁡(λ1,λ2,…,λn).

If all eigenvalues are real, they can be increasingly ordered. Λ↑ is the eigenvalue matrix of increasingly ordered eigenvalues.

τ is a permutation of {1,2,…,n}.

A~=A+ΔA is a perturbed matrix, where ΔA is perturbation. (λ~,x~) are the eigenpairs of A~.

Condition number of a nonsingular matrix X is κ(X)=∥X∥∥X−1∥.

Let X,Y∈Cn×k with rank⁡(X)=rank⁡(Y)=k. The canonical angles between their column spaces, θi, are defined by cos⁡θi=σi, where σi are the singular values of the matrix

(Y∗Y)−1/2Y∗X(X∗X)−1/2.

The canonical angle matrix between X and Y is

Θ(X,Y)=diag⁡(θ1,θ2,…,θk).

Facts

Bounds become more strict as matrices have more structure. Many bounds have versions in spectral norm and Frobenius norm. For more details and the proofs of the Facts below, see R.-C. Li, Matrix Perturbation Theory, pp 21.1-21.20, and the references therein.

  1. There exists τ such that

∥Λ−Λ~τ∥2≤4(∥A∥2+∥A~∥2)1−1/n∥ΔA∥21/n.

  1. First-order perturbation bounds. Let (y,λ,x) be an eigentriplet of a simple λ. ΔA changes λ to λ~=λ+δλ, where

δλ=y∗(ΔA)xy∗x+O(∥ΔA∥22).

  1. Let μ be a semisimple eigenvalue of A with multiplicitiy k, and let X,Y∈Cn×k be the matrices of the corresponding right and left eigenvectors, that is, AX=λX and Y∗A=λY∗, such that Y∗X=Ik. ΔA changes the k copies of μ to μ~=μ+δμi, where δμi are the eigenvalues of Y∗(ΔA)X up to O(∥ΔA∥22).

  2. Perturbations and the inverse: if ∥A∥p<1, then I−A is nonsingular and (for proof see GVL p. 74 (98))

(I−A)−1=∑k=0∞Ak,

∥(I−A)−1∥p≤11−∥A∥p,∥(I−A)−1−I∥p≤∥A∥p1−∥A∥p.

  1. Geršgorin Circle Theorem. If X−1AX=D+F, where D=diag⁡(d1,…,dn) and F has zero diagonal entries, then (for proof see GVL p.357 (381))

σ(A)⊆⋃i=1nDi,

where

Di={z∈C:|z−di|≤∑j=1n|fij|}.

Moreover, by continuity, if a connected component of D consists of k circles, it contains k eigenvalues.

maximinj|λ~i−λj|≤∥X−1(ΔA)X∥p≤κp(X)∥ΔA∥p.

  1. If A and A~ are diagonalizable, there exists τ such that

∥Λ−Λ~τ∥F≤κ2(X)κ2(X~)∥ΔA∥F.

If Λ and Λ~ are real, then

∥Λ↑−Λ~↑∥2,F≤κ2(X)κ2(X~)∥ΔA∥2,F.

  1. If A is normal, there exists τ such that ∥Λ−Λ~τ∥F≤n∥ΔA∥F.

  2. Hoffman-Wielandt Theorem. If A and A~ are normal, there exists τ such that ∥Λ−Λ~τ∥F≤∥ΔA∥F.

  3. If A and A~ are Hermitian, for any unitarily invariant norm ∥Λ↑−Λ~↑∥≤∥ΔA∥. In particular,

maxi|λi↑−λ~i↑|≤∥ΔA∥2,∑i(λi↑−λ~i↑)2≤∥ΔA∥F.

  1. Residual bounds. Let A be Hermitian. For some λ~∈R and x~∈Cn with ∥x~∥2=1, define residual r=Ax~−λ~x~. Then |λ~−λ|≤∥r∥2 for some λ∈σ(A).

  2. Let, in addition, λ~=x~∗Ax~, let λ be closest to λ~ and x be its unit eigenvector, and let

η=gap⁡(λ~)=minλ≠μ∈σ(A)|λ~−μ|.

If η>0, then

|λ~−λ|≤∥r∥22η,sin⁡θ(x,x~)≤∥r∥2η.

max1≤j≤k|μj−λij|≤∥R∥2σmin(X),∑j=1k(μj−λij)2≤∥R∥Fσmin(X).

(The indices ij need not be the same in the above formulae.)

  1. If, additionally, X∗X=I and M=X∗AX, and if all but k of A's eigenvalues differ from every one of M's eigenvalues by at least η>0, then

∑j=1k(μj−λij)2≤∥R∥F2η1−∥R∥F2/η2.

  1. Let A=[ME∗EH] and A~=[M00H] be Hermitian, and set η=min|μ−ν| over all μ∈σ(M) and ν∈σ(H). Then

max|λj↑−λ~j↑|≤2∥E∥22η+η2+4∥E∥22.

[X1∗X2∗]A[X1X2]=[A1A2],[X1X2]unitary,X1∈Cn×k.
